# Interkosmos System

NixOS with GitOps orientation with Docker based build system.

## Installation

1. Install Docker
2. Clone repository

## Usage

### Build ISO image

```bash
./build.sh iso
``` 

### Build Scaleway image

```bash
./build.sh scaleway
```

### Access build container

```bash
./build.sh bash
```
